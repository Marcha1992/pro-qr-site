Описание проекта:
Проект «Учебно-методический автоматизированный комплекс «Установка производства водорода» представляет собой уникальное учебное пособие для подготовки технических специалистов. Данный комплекс включает в себя масштабную модель установки (1:87) размером 1200 на 900 мм, позволяющую визуализировать процесс производства водорода. Модель обеспечивает возможность запуска и управления установленными элементами в смоделированных условиях, что позволяет обучающимся глубже понять функционирование технологического процесса.
В составе комплекса находятся различные модели, отражающие материальную копию элементов установки по производству водорода.
Физическая модель оборудования эстакады и несущих конструкций смоделирована в программе Компас 3D и распечатана на 3D принтере:
1. Общезаводской факел — высота 700 мм. Устройство предназначено для безопасного сжигания избыточного газа.
2. Печь риформинга — высота 400 мм, выполняет процесс парового риформинга, преобразуя природный газ в водород и углекислый газ.
3. Межцеховая эстакада — высота 250 мм для удобного доступа к технологическим трубопроводам и оборудованию.
4. Компрессорный блок закрытого типа — высота здания 170 мм, отвечает за сжатие водорода для его последующего хранения и транспортировки.
5. Блок этажерки АВО — высота 200 мм, важен для распределения и контроля технологического оборудования.
6. Межцеховая эстакада факельного коллектора — соединяет компоненты газоотведения и сжигания.
7. Секция КЦА предназначена для выделения водорода из синтез-газа методом короткоцикловой адсорбции.
8. Имитация процесса преобразования природного газа в водород осуществляется путем циркуляции в системе технологических трубопроводов специального глицерина, который служит модельной рабочей средой.
Процессы установки управляются посредством разработанной на основе SCADA-систем интегрированного программного обеспечения на базе веб интерфейса на языке программирования Java, для корректной работы интерфейса на АРМ оператора установлен виртуальный сервер с разработанной базой данных для обмена данными по беспроводной сети передачи данных посредством WI-FI с контроллером. Данный интерфейс позволяет пользователям не только наблюдать за функционированием установки, но и в реальном времени взаимодействовать с моделью, изменяя параметры, такие как работа факела, включение компрессорных установок и функционирование запорно-регулирующей арматуры.
